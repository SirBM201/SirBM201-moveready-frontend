# MoveReady Functions and User Benefits

## Simple description

MoveReady is a relocation planning assistant. It helps people check a route, understand the risk, prepare documents, estimate costs, save their plan, set reminders, and ask for support before spending serious money.

It is not a visa approval service. It is not a government website. It is not a job approval service. It gives planning guidance only.

## Main functions

### 1. Start guide

The Start page tells a new user what to do first, second, third, and fourth.

Benefit: the user does not feel lost when opening the app.

### 2. Account and active profile

The Account page lets a user save one active relocation profile with their name, email, current country, target country, route type, family count, timeline, and available funds.

Benefit: the user does not need to enter the same details again and again.

### 3. Profile cleanup

The Account page lets a user choose the correct active profile and hide old test profiles.

Benefit: the app uses the correct profile for route checks, reports, alerts, saved routes, timeline, and support.

### 4. Route checker

The Route Checker uses the active profile or manual details to generate a starter readiness result.

Benefit: the user can check if a route looks realistic before paying agents, booking services, or preparing expensive documents.

### 5. Readiness report

The report shows route summary, readiness score, risk label, document checklist, budget estimate, source status, and next steps.

Benefit: the user gets a clear plan instead of scattered information from many websites and agents.

### 6. Saved reports

The Reports page helps the user find reports later by signed-in account, report reference, email, or phone.

Benefit: the user can return to the plan without starting again.

### 7. Saved routes

The Saved Routes page stores serious route ideas and allows the user to revisit them later.

Benefit: the user can compare options and avoid forgetting important plans.

### 8. Alerts and watchlist

The Alerts page stores reminders for route openings, closing dates, result windows, eligibility changes, document updates, fee changes, and report refresh reminders.

Benefit: the user can remember what to check without depending only on agents or social media posts.

### 9. Timeline

The Timeline page records dated actions such as document preparation, appointments, results, follow-up, and deadlines.

Benefit: the user can manage the process step by step.

### 10. Services and support requests

The Services page allows a user to request help such as document review, expert review, courier, legalization, translation, accommodation, pickup, or settlement help.

Benefit: the user can ask for support after understanding their route. The request stays private and should be reviewed before any provider handoff.

### 11. Opportunities

The Opportunities page tracks official opportunity routes such as visa lotteries, ballots, invitation pools, quotas, scholarships, and application windows.

Benefit: the user can monitor opportunities more safely and avoid fake announcements.

### 12. Admin review

The admin area helps MoveReady review profiles, reports, support requests, alerts, partner applications, and user records before taking further action.

Benefit: the platform stays safer, more controlled, and more trustworthy.

## Best user journey

1. Start.
2. Save or load one profile in Account.
3. Check route.
4. Generate report.
5. Read next steps.
6. Save route or create alert.
7. Ask for support only when needed.

## Main user benefits

- Saves time.
- Reduces confusion.
- Reduces scam risk.
- Helps users know what to prepare.
- Helps users estimate costs.
- Helps users track important route changes.
- Keeps reports and records connected to one account.
- Gives a clear next-step plan.
- Makes relocation planning easier for non-expert users.

## Important safety promise

MoveReady must always explain that it is advisory guidance only. Users must still confirm requirements from official sources before submitting applications, paying providers, booking travel, or making final decisions.
